# SauvetageTeam — Guide du responsable d'équipe

## Vue d'ensemble

Le responsable d'équipe (coach) inscrit les athlètes de son club pour la compétition à venir. Ce rôle a accès aux onglets **Inscriptions individuelles** et **Inscriptions relais**, limité aux athlètes de son propre club.

**Votre rôle dans le cycle de compétition :** Vous êtes notifié à l'étape ③ (courriel d'invitation avec le NIP de votre club). Vous agissez à l'étape ④ (inscrire les athlètes avant la date limite). Une fois la compétition terminée et les résultats importés par l'organisateur, votre NIP est régénéré — vous aurez besoin d'une nouvelle invitation pour le prochain meet.

---

## Obtenir votre NIP

### Par courriel d'invitation

1. L'organisateur envoie un courriel d'invitation à l'adresse enregistrée de votre club
2. Cliquer sur le **lien sécurisé** dans le courriel — il révèle le NIP de votre club (usage unique)
3. Noter le NIP — vous l'utiliserez pour vous connecter

### Par auto-invitation

Si vous n'avez pas reçu d'invitation :
1. Aller sur la page de connexion et cliquer **Demander une invitation**
2. Sélectionner votre club, confirmer le courriel, cliquer **Envoyer l'invitation**
3. Vérifier votre courriel pour le lien sécurisé

---

## Connexion

1. Ouvrir l'application SauvetageTeam dans un navigateur
2. Entrer le **NIP de votre club**
3. Cliquer **Connexion** — la barre de titre affiche le nom de votre club

## Inscription

### Onglet Inscriptions individuelles

Après la connexion, les athlètes de votre club s'affichent dans l'onglet **Inscriptions individuelles** :
- Athlètes regroupés par **catégorie d'âge** et **genre**
- Chaque athlète affiche son nom, son statut d'inscription et le nombre d'épreuves inscrites
- Utiliser la **barre de recherche** pour trouver un athlète

### Inscrire aux épreuves

1. Cliquer sur le nom d'un athlète pour déplier son panneau d'inscription
2. **Cocher la case** à côté de chaque épreuve pour inscrire — décocher pour désinscrire
3. Si éligible à plusieurs catégories (15-18 / Open / Masters), sélectionner dans le **menu déroulant**

### Temps d'inscription et meilleurs temps

- Les **meilleurs temps** (bassin 50m et 25m) sont affichés en lecture seule
- Le **temps d'inscription** est pré-rempli à partir du meilleur temps correspondant au bassin
- Vous pouvez **ajuster le temps d'inscription** si nécessaire

### Épreuves de relais

Les relais sont gérés dans l'onglet **Inscriptions relais** :

1. Cocher l'épreuve de relais pour inscrire votre club
2. Cliquer **Assigner les membres** pour définir les positions (1er, 2e, 3e, 4e)
3. Les positions peuvent être laissées vides et remplies plus tard

---

## Clôture des inscriptions

- La **date limite** est fixée par l'organisateur
- Après la date limite, le formulaire d'inscription devient **en lecture seule**
- Vous pouvez toujours consulter vos inscriptions mais ne pouvez plus les modifier

---

## Page publique des meilleurs temps

Une page publique est disponible (sans connexion) pour consulter les meilleurs temps :
- Depuis la page de connexion, cliquer **Consulter les meilleurs temps**
- Rechercher par nom d'athlète ou club

---

## Conseils

- Inscrire les athlètes le plus tôt possible — ne pas attendre la date limite
- Vérifier que les temps d'inscription sont raisonnables
- Si le meilleur temps d'un athlète semble erroné, contacter l'administrateur
- Les positions de relais peuvent être modifiées jusqu'à la date limite
- Si vous perdez votre NIP, utilisez le flux d'auto-invitation

## Référence rapide

| Action | Comment |
|--------|---------|
| Se connecter | Entrer le NIP du club → Connexion |
| Inscrire à une épreuve | Cliquer sur l'athlète → Cocher l'épreuve |
| Changer de catégorie | Sélectionner dans le menu déroulant |
| Ajuster le temps d'inscription | Cliquer sur le champ de temps, taper le nouveau temps |
| Assigner les relayeurs | Cocher l'épreuve de relais → Assigner les membres |
| Consulter les meilleurs temps | Page de connexion → Consulter les meilleurs temps |
| Obtenir un nouveau NIP | Page de connexion → Demander une invitation |
